# Execution in dev environment

1. Clone this repository.
2. Execute the following command

```bash
yarn install
```

3. Must have Nest CLI installed

```bash
npm i -g @nestjs/cli
```

4. Running up the database

```bash
docker-compose up -d
```


## Stack used
* MongoDB
* NestJS
